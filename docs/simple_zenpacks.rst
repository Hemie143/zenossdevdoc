Simple ZenPacks
===============

********
Creation
********

* Using the Zenoss Gui: "Add to ZenPack"
* Items:

   * Device Classes
   * Event Classes
   * Event Mappings
   * User Commands
   * Event Commands
   * MIBs
   * Service Classes
   * Process Classes
   * Device Organizers
   * Performance Templates

* During Export, all objects under "ZenPack Provides" are written to objects.xml
